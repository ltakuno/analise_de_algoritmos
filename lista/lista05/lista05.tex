\item (Partition - Education Codeforces Round 39 [Rated div. 2]) 

Dada uma sequência $a$ consistindo de $n$ inteiros. É possível particionar esta sequência em duas sequências $b$ e $c$ de tal maneira que todo elemento pertence a exatamente a uma destas sequências.

Seja $B$ a soma dos elementos de $b$, e $C$ a soma dos elementos de $c$ (Se alguma dessas sequências é vazia então a soma é 0). Qual é o valor máximo de $B - C$?

\textbf{Entrada:} A primeira linha contém um inteiro $n$ ($1 \leq n \leq 100$) - o número de elementos de $a$.

A segunda linha contém $n$ inteiros $a_1, a_2, \cdots, a_n$ $(-100 \leq a_i \leq 100)$ os elementos da sequência $a$.

\textbf{Saída:} a impressão do valor máximo de $B-C$, onde $B$ é a soma dos elementos de $b$, e $C$ é a soma dos elementos de $c$.

Exemplo1:

\begin{verbatim}
Entrada:
3
1 -2 0

Saída
3
\end{verbatim}

Exemplo2:

\begin{verbatim}
Entrada:
6
16 23 16 15 42 8

Saída
120
\end{verbatim}

\textbf{Nota:}

No primeiro exemplo podemos escolher $b = \{1,0\}, c = \{-2\}$. Então $B=1$, $C=-2$, $B-C = 3$.

No segundo exemplo podemos escolher $b = \{16, 23, 16, 15, 42, 8\}, c = \{\}$. Então $B=120$, $C=0$, $B-C = 120$.

\end{enumerate}
