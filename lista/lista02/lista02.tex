\begin{enumerate}
\item Prove a corretude do algoritmo de Euclides, que determina o Máximo Divisor Comum (MDC) entre dois números naturais. 
{\blue
\begin{verbatim}
mdc(m,n)
1  x = m
2  y = n
3  enquanto y != 0 faça
4    r = x mod y
5    x = y
6    y = r
7  devolva x
\end{verbatim}
}

\item Demonstre a corretude do algoritmo a seguir, que converte um número inteiro na base decimal para a sua representação em binário, ou seja, com 0's e 1's.

{\blue
\begin{verbatim}
converte(n)
1  t = n
2  k = -1
3  enquanto t > 0 faça
4    k = k + 1
5    b[k] = t mod 2
6    t = t div 2
7  devolva b
\end{verbatim}
}

\item Descreva (em pseudocódigo) a versão recursiva da técnica de busca binária para um \textit{array} ordenado e mostre a corretude deste algoritmo.

\item Use o método de árvore de recursão para determinar o tempo de execução dos algoritmos expressos pelas recorrências abaixo:

\begin{enumerate}
\item $T(n) = 3 T(\lfloor n/2 \rfloor) + \Theta(n)$
\item $T(n) = 4 T(\lfloor n/2 \rfloor) + \Theta(n)$
\item $T(n) = 2 T(\lfloor n/2 \rfloor) + \Theta(n^2)$
\item $T(n) = 2 T(\lfloor n/2 \rfloor) + \Theta(n)$ (Mergesort)
\item $T(n) = 2 T(\lfloor n/3 \rfloor) + \Theta(n)$ 
\end{enumerate}

\end{enumerate}
